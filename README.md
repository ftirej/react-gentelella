# react-gentelella

Gentelella theme port to React.
This project applies the [semantic version 2.0.0](http://semver.org) specification.

## USAGE
THIS PROJECT IS UNDER INITIAL DEVELOPMENT.
DO NOT USE IT ON PRODUCTION PROJECTS.
